# Kazzi-Hub Projects

The Kazzi-Hub platform has been split into multiple independent repositories for better separation of concerns.

## Repositories

- **kazzihub.website**: The public-facing website.
- **kazzihub.student.portal**: The learning portal for students.
- **kazzihub.admin.portal**: The internal dashboard for staff.
- **kazzihub.core**: The core backend service.

## Tech Stack

This section outlines the technology stack used across the Kazzi-Hub platform.

### Frontend

- **Framework:** Next.js
- **Deployment:** Vercel

### Backend (API)

- **Framework:** Python FastAPI
- **Containerization:** Docker
- **Deployment:** Supabase

### Infrastructure & Services

- **Database:** PostgreSQL (hosted on Supabase)
- **Authentication:** Supabase Auth
- **Storage:** Supabase Storage
