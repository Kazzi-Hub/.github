# Tech Stack

This document outlines the technology stack used across the Kazzi-Hub web application.

## Frontend

- **Framework:** Next.js
- **Deployment:** Vercel

## Backend (API)

- **Framework:** Python FastAPI
- **Containerization:** Docker
- **Deployment:** Supabase

## Infrastructure & Services

- **Database:** PostgreSQL (hosted on Supabase)
- **Authentication:** Supabase Auth
- **Storage:** Supabase Storage
